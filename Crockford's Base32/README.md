# Crockford's Base32
Написать программу, которая осуществляет кодирование и декодирование информации в Base32.
* Алфавит: `0123456789ABCDEFGHJKMNPQRSTVWXYZ`
* Терминальный символ: `=`
* При декодировании учитывать, что символы `iIlL` – это символ `1`, а символы `оО` – это символ `0`.
* При декодировании учитывать, что большие и маленькие буквы не различаются.
* При декодировании учитывать, что сообщение может быть разбито на строки (символы в сообщении, которые не входят в алфавит Base32 необходимо игнорировать).

Например, сообщение `DoSal0iL` эквивалентно `D0SA1011`.

## Пример
Исходное сообщение: `Hello, world!`

Сообщение в Base32: `91JPRV3F5GG7EVVJDHJ22===`

## Требования к реализации

Базовый синтаксис команды выглядит следующим образом:
```
base32 [параметр] имя-входного-файла имя-выходного-файла
```

* Название програмы: `base32`
* Параметр `-e` — позволяет осуществлять кодирование в Base32.
* Параметр `-d` — позволяет осуществлять декодирование из Base32.

Программа не должна использовать контейнеры STL (std::vector, std::string и т. д.).
