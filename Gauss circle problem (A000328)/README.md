# Точки в окружности
Дана окружность с целочисленным радиусом R (0 < R < 2'000'000'000). 

Посчитать и вывести на экран число точек с целочисленными координатами, лежащих внутри окружности и на границе окружности.

## Вход
Целое число больше 0 и меньше 2000000000 — радиус окружности.

## Выход
Целое число — количество точек с целочисленными координатами, лежащих внутри окружности или на окружности.
